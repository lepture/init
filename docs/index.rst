Init
====

This is a python version of init_ project.


Installation
------------

Install the commandline tools with pip::

    $ pip install init

Alternatively, use ``easy_install``::

    $ easy_install init


Usage
-----

Find the help menu with::

    $ init -h

Let's create a python module for instance::

    $ mkdir foo
    $ cd foo
    $ init python


Template
--------

The offical templates are available on github at: https://github.com/init-template .

You can also create a template under your own namespace, for example, lepture/foo. And init with ``init lepture/foo``.
